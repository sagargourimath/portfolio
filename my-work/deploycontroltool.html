Code Deployment Control Deploy Control Tool

To control flex deployments

A simple application to control deployments on flex (Linux) servers. This is achieved by modifying the /etc/hosts.allow file. A console application is used to unlock servers during daily scheduled windows, daylight saving considered when unlocking environments where it is to be considered.

A manual option is also available so that authorized users could lock/unlock based on the requirement.
